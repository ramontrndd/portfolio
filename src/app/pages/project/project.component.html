Github Deploy Cafe-Management Projeto Front-End

Cafe-Management - Front-End: é um projeto desenvolvido com Angular, projetado para facilitar a gestão de cafeterias. Com um layout responsivo e intuitivo, oferece uma experiência ao usuário em qualquer dispositivo, Utilizando SCSS para estilos, o projeto garante uma apresentação visualmente atraente e personalizável.

Github Deploy Cafe-Management Projeto Back-End

Cafe-Management - Back-End: Desenvolvi API's utilizando Node.js e Express, conectadas a um Banco de Dados MySQL. As API's abrangem usuários, produtos, contas e um dashboard, possibilitando diversas interações com o banco de dados, incluindo operações CRUD para uma gestão eficiente.
